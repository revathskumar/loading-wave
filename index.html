<loading-wave>

Custom Element for Gravatar without polymer.

Usage

  1. Install the component using Bower:

    $ bower install loading-wave --save

  2. Import Custom Element:

    <link rel="import" href="bower_components/loading-wave.html">

  3. Start using it!

    <loading-wave email="youremail@host.com" size="200"></loading-wave>
